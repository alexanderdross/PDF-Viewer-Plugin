## Usage

These markdown files are designed to be:

1. **Readable on GitHub** - Full markdown formatting with tables, code blocks, etc.
2. **Convertible to HTML** - Can be processed by static site generators (Hugo, Jekyll, etc.)
3. **Embeddable in CMS** - Content can be copied to WordPress/Drupal pages

## Styling Notes

- All pages include the product logo from the website CDN
- Tables use GitHub-flavored markdown
- Code blocks use syntax highlighting
- Links are relative where possible, absolute for external resources
