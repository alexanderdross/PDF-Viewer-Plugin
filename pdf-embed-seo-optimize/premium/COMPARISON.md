## Pro+ Enterprise Exclusive Features

Pro+ Enterprise includes all Pro features plus these enterprise-grade capabilities:

### Advanced Analytics
- **Heatmaps**: Visualize where users spend time on each page
- **Engagement Scoring**: Automatic scoring based on views, time spent, and interactions
- **Geographic Analytics**: Track document views by country and region
- **Device Analytics**: Detailed breakdown by device type, browser, and OS

### Enterprise Security
- **Two-Factor Authentication (2FA)**: TOTP-based 2FA for sensitive documents
- **IP Whitelisting**: Restrict access to specific IP addresses or ranges
- **Audit Logs**: Complete audit trail with configurable retention (up to 365 days)
- **Failed Attempt Limiting**: Automatic lockout after configurable failed attempts

### Document Versioning
- **Version History**: Keep up to 10 versions of each document
- **Auto-Versioning**: Automatic version creation on file replacement
- **Version Comparison**: Compare changes between versions
- **Version Restore**: Restore any previous version

### Annotations & Signatures
- **PDF Annotations**: Highlight, underline, strikethrough, and comment
- **User Annotations**: Allow authenticated users to add their own annotations
- **Digital Signatures**: E-signature capability for document signing
- **Annotation Export**: Export annotated PDFs

### Compliance & Privacy
- **GDPR Mode**: Full GDPR compliance with consent tracking and data export
- **HIPAA Mode**: HIPAA-compliant logging and access controls
- **Data Retention Policies**: Configurable automatic data cleanup
- **Consent Management**: User consent tracking and audit

### Webhooks & Integrations
- **Webhook Events**: Trigger on view, download, password attempt, annotation, signature
- **Webhook Security**: HMAC signature verification for secure delivery
- **Retry Logic**: Automatic retry with exponential backoff
- **Event Filtering**: Select specific events to send

### White Label & Branding
- **Custom Branding**: Replace all PDF Embed branding with your own
- **Custom Logo**: Display your logo in the viewer
- **Hide Powered By**: Remove all attribution
- **Custom CSS**: Inject custom styles for complete customization

## Which Tier Is Right For You?

### Choose **Free** if you:
- Need basic PDF embedding on your site
- Want SEO-optimized PDF pages
- Don't need analytics or access control
- Are building a simple documentation site

### Choose **Pro** if you:
- Need to organize PDFs with categories and tags
- Want detailed analytics and view tracking
- Need password protection for sensitive documents
- Want to generate leads with email gates
- Need to sell PDFs with WooCommerce

### Choose **Pro+ Enterprise** if you:
- Need enterprise-grade security (2FA, IP whitelisting, audit logs)
- Require HIPAA or strict GDPR compliance
- Want advanced analytics with heatmaps and engagement scoring
- Need document versioning and version history
- Require PDF annotations or digital signatures
- Want webhook integrations with external systems
- Need white-label branding for client sites
- Require dedicated support with SLA

## Getting Started

### Free Version
1. Download from WordPress.org or GitHub
2. Upload and activate the plugin
3. Start creating PDF Documents

### Pro Version
1. Purchase a Pro license from our website
2. Download the premium package
3. Upload the `premium` folder to your plugin directory
4. Activate your license in PDF Documents > License
5. Configure premium features in PDF Documents > Settings > Premium

### Pro+ Enterprise Version
1. Purchase a Pro+ Enterprise license
2. Download the Pro+ package (includes Free + Pro + Pro+)
3. Upload both `premium` and `pro-plus` folders
4. Activate both licenses (Pro and Pro+)
5. Configure Pro+ features in PDF Documents > Pro+ Settings
